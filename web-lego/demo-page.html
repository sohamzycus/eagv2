📋 How to Demo Web Lego

1

Activate Extension

Click the Web Lego icon in your browser toolbar and press "Activate Web Lego"

2

Select Elements

Click "Select" in the floating toolbar, then hover and click on elements below

3

Add to Canvas

Click "Add Selected" to move elements to your canvas, or use "Add All" for everything

4

Create & Export

Open canvas, arrange blocks, edit content, and export as HTML or share links

🎯

Easy Selection

This card demonstrates perfect element selection. The colorful header, text content, and overall styling make it ideal for showcasing the selection feature.

  • Hover highlights in blue
  • Click to select (turns green)
  • Visual feedback with checkmarks
  • Multiple selections supported
✏️

Editable Content

Once in the canvas, all content becomes editable! Double-click any text to modify it, creating entirely new layouts from existing content.

  • Double-click to edit text
  • Maintain original styling
  • Mix webpage + custom content
  • Real-time editing
🎨

Creative Layouts

Drag, resize, and arrange elements freely. Create overlapping compositions, adjust sizes, and build professional layouts that look amazing.

  • Free-form positioning
  • Resize with corner handles
  • Overlap for complex designs
  • Professional canvas tools
📤

Export & Share

Export your creations as complete HTML files, copy the code, or generate shareable links. Your layouts become real, usable web content!

  • Download HTML files
  • Copy HTML/CSS code
  • Share via links
  • Save as templates
